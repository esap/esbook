# 1. 菜鸟科技-ES

### 新手入门
欢迎来到ES的世界，让我们开始探索吧！

ES全称：ExcelServer，是一种利用Excel来快速设计企业级应用系统的平台，目前主要包括勤哲，慧表，活字格等产品。

ES的一些版本存在许多坑,这里主要列出了一些相关的重要坑：

- 修改字段名可能会保存失败。
- 回写公式可能会失败并且无任何提示（也可能是网络问题）。
- 自动编号跳号。
- 逗逼的dept admin角色。
- 超链接存在权限缺陷。
- 工作流会卡在进行中待办。
- 用2007+版excel设计模板会导致颜色不兼容，vba追加失败等问题。
- [了解更多...](http://ylin.wang/2014/10/31/EsDiediedie/)
